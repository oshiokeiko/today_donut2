# 今日のおすすめドーナツ占い

## どんなアプリか
今日の気分を選ぶと、オリジナルドーナツの占い結果とラッキーアイテム・ラッキーカラーを教えてくれるアプリです。

## 使い方
`donut_fortune.html` をブラウザで開くだけで動きます。

## アプリの特徴

- **気分を3択で選べる**：絶好調・ふつう・ちょっとつかれた の3ボタン
- **オリジナルドーナツが10種類**：それぞれに占いメッセージつき
- **レアドーナツが約5%で出現**：「ゴールデン・ナッツ・レガリア」が出ると紙吹雪🎊
- **ラッキーアイテム＆カラーをランダム表示**：15種類の中からその日の運勢をお知らせ
- **結果カードがふわっとフェードイン**：モーダルの表示・非表示にアニメーションつき

## 進化させた点
- ドーナツデータをオブジェクト配列にまとめて管理しやすくした
- 確率制御によるレアドーナツの実装（通常は均等、レアのみ5%）
- 3つのボタン処理を1つにまとめてコードをシンプルに
- フェードイン・アウトアニメーションの追加
- レア出現時の紙吹雪エフェクト（canvas-confetti）

## 使った技術
- HTML / Tailwind CSS
- JavaScript / jQuery
- canvas-confetti（紙吹雪ライブラリ）
