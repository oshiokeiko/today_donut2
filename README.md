# 今日のおすすめドーナツ占い

## プロダクトの紹介
今日の気分を選ぶと、オリジナルドーナツの占い結果とラッキーアイテム・ラッキーカラーを教えてくれるアプリです。気分は「絶好調・ふつう・ちょっとつかれた」の3択で、全11種類のドーナツの中からその日の1枚が出ます。約5%の確率でレアドーナツが出現します。

## 使い方
`donut_fortune.html` をブラウザで開くだけで動きます。

## アプリの特徴
- **気分を3択で選べる**：絶好調・ふつう・ちょっとつかれた の3ボタン
- **オリジナルドーナツが10種類**：それぞれに占いメッセージつき
- **レアドーナツが約5%で出現**：「ゴールデン・ナッツ・レガリア」が出ると紙吹雪🎊
- **ラッキーアイテム＆カラーをランダム表示**：15種類の中からその日の運勢をお知らせ
- **結果カードがふわっとフェードイン**：モーダルの表示・非表示にアニメーションつき

## 工夫した点・こだわった点
- 前回はif文の中でドーナツを直接書いていたが、今回はオブジェクト配列にまとめて管理しやすくした
- 出てくるドーナツに確率差をつけ、レアドーナツの出現確率を5%に設定した
- 3つのボタン処理を1つにまとめ、コードの重複をなくした
- フェードイン・アウトアニメーションと紙吹雪エフェクトで演出を追加した

## 苦戦した点・共有したいハマりポイント
- **重み付けの理解が難しかった**：最初はweightプロパティで各ドーナツに数値を持たせる方法を試みたが、概念の理解に時間がかかった。最終的に `RARE_CHANCE = 5`（5%）という直感的な書き方に切り替えたことでスッキリ理解できた
- **for文の`i`が何者かわからなかった**：`for (let i = 0; i < donuts.length; i++)` の `i` が「何番目か」を示す番号だと気づくまで時間がかかった
- **`total = total + ...` の意味がわからなかった**：右辺を先に計算してから左辺に入れ直す、という考え方がなかなか腑に落ちなかった
- **`console.log` のスペース**：`console log(donuts)` とスペースを入れてしまいエラーになった。ドットが必要
